# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please use [GitHub Security Advisories](https://github.com/y-miyazaki/arc/security/advisories/new) to report the vulnerability privately.

You can expect:

- An acknowledgment within 48 hours
- A status update within 7 days
- A fix or mitigation plan for confirmed vulnerabilities

## Scope

This policy covers the arc CLI tool and its release artifacts. AWS credential handling is delegated to the AWS SDK and is outside the scope of this project.
